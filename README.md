Simple Calculator
A basic web-based calculator built with HTML, CSS, and JavaScript. Perform arithmetic operations like addition, subtraction, multiplication, and division. Designed for simplicity and responsiveness.

Features
Clean, modern UI with a dark theme.
Supports basic math operations (addition, subtraction, multiplication, division).
Error handling for invalid inputs (e.g., division by zero).
Responsive design for desktop and mobile devices.
Easy-to-use interface with clickable buttons.


View it live on GitHub Pages: https://develrags.github.io/calculator-project

Usage
Click the number and operator buttons to build your expression.
Press = to calculate the result.
Press C to clear the display.
Example: Click 5, +, 3, =, and the display shows 8.
Technologies Used
HTML: Provides the structure of the calculator.
CSS: Handles styling and responsive layout.
JavaScript: Manages user interactions and calculations.
